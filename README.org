* PlusHC.jl

PlusHC.jl is a simple little utility offering a single export: ~hc~
(standing for hermitian conjugate), whose only purpose is to turn the
expression ~x + hc~ into ~x + x'~.

#+BEGIN_SRC julia
julia> using PlusHC

julia> (1 + im) + hc
2
